# 🧠 Cortexia — AI에게 인간형 두뇌를 달아주세요

AI가 대화할 때마다 모든 걸 잊어버리는 거, 느끼셨죠?

코딩 스타일을 설명하고, 프로젝트 구조를 알려주고, 선호하는 방식을 전달합니다. 그런데 세션이 끝나면? 다음에 또 처음부터 다시 설명해야 합니다. 매번 수천 토큰이 낭비됩니다. 시간도. 돈도.

**Cortexia가 해결합니다.**

계산 신경과학에서 사용하는 SNN(스파이킹 신경망)을 기반으로, AI에게 진짜 두뇌를 달아줍니다. 데이터베이스가 아닙니다. 두뇌입니다.

- 반복하면 기억이 강화되고, 시간이 지나면 자연스럽게 희미해집니다 (에빙하우스 망각곡선)
- 신경전달물질 시뮬레이션 (도파민, 세로토닌, 노르에피네프린)
- 대화를 통해 AI만의 성격이 형성됩니다
- 스마트 문서 도서관 — 파일을 넣으면 두뇌가 도서관처럼 검색합니다
- 토큰 절약 추적기 — 얼마나 절약했는지 정확히 보여줍니다

기존 RAG와는 다릅니다. RAG는 벡터 데이터베이스를 검색합니다 — 차갑고, 기계적이고, 정적입니다. Cortexia는 인간처럼 기억합니다. 중요한 건 남고, 사소한 건 사라집니다.

**Claude Code** 및 **MCP** (Model Context Protocol)를 통해 모든 LLM과 호환됩니다.

```
npm install -g cortexia
```

## 절약 계산

Cortexia 없이: 50 세션 × 반복 설명 500 토큰 = **25,000 토큰 낭비**

Cortexia 사용: 한 번 설명하면 두뇌가 기억합니다. **~24,500 토큰 절약.**

대규모 사용 시 — 하루 100 세션 × 30일 = **월 1,500,000 토큰 낭비.** Cortexia가 스스로 비용을 회수합니다.
